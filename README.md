# Práctica 2: Análisis de Respuesta en Frecuencia de un Circuito RC (Filtro Pasa Bajas)

## 📌 Descripción del Proyecto
Este repositorio contiene la simulación computacional de un circuito RC en configuración de filtro pasa bajas.El objetivo principal es analizar su respuesta en frecuencia y validar teóricamente el comportamiento de atenuación y desfasamiento para caracterizar su funcionamiento como sistema dinámico de primer orden. 

El modelado matemático y la visualización interactiva (osciloscopio simulado y Diagrama de Bode) se desarrollaron íntegramente en Python.

## ⚙️ Parámetros del Circuito Simulado
**Resistor (R):** 200 Ω
**Capacitor (C):** 8.5 µF

## 🚀 Tecnologías Utilizadas
* **Python 3.x**
* **NumPy:** Cálculo vectorizado de la función de transferencia, reactancia capacitiva y modelado de señales en el dominio del tiempo.
* **Matplotlib:** Generación de la interfaz gráfica interactiva (osciloscopio) y renderizado del Diagrama de Bode.

## 📁 Estructura del Repositorio
* `simulacion_filtro_rc.py`: Script principal que ejecuta la simulación de forma interactiva.
